# Prev Next Link

前と次のリンクのテキストをカスタマイズすることができます。サイドバーに表示しているテキストとは異なるテキストを前/次のリンクに表示したい場合に便利です。

## prev

- Type: string
- Details: 前ページへのリンクに表示するテキストを指定します。frontmatterで設定しない場合、テキストはサイドバーの設定から推測される。
- Example:

```yaml
---
prev: 'Get Started | Markdown'
---
```

## next

- Type: string
- Details: prevと同じで、次のページを表示します。
