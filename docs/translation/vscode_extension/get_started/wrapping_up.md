# 仕上げ

「はじめての拡張機能」トピックでは、拡張機能を作成、実行、デバッグする方法を学びました。拡張機能の解剖トピックでは、Visual Studio Code 拡張機能の開発における基本的な概念を学びました。しかし、まだ氷山の一角に過ぎません。ここでは、VS Code 拡張機能の開発スキルをさらに向上させるための推奨ルートをいくつか紹介します。

## エクステンションの機能

このセクションでは、VS Code API と Contribution Points をいくつかのカテゴリに分け、それぞれで拡張機能が実現できることを簡単に説明します。拡張機能のアイデアが実現可能かどうか、VS Code APIを確認したり、新しい拡張機能のアイデアを得るために拡張機能のセクションを読んだりして、検証してください。

## ガイドとサンプル

私たちは、あなたが適応できる素晴らしいサンプル拡張機能のコレクションを用意しており、それらのいくつかはソースコードを説明する詳細なガイドを含んでいます。すべてのサンプルとガイドは Extension Guide Listing もしくは vscode-extension-samples リポジトリで見つけることができます。

## UX ガイドライン

あなたの拡張機能を VS Code のユーザーインターフェイスに違和感なくフィットさせるために、UX ガイドラインを参照してください。ここでは、拡張機能の UI を作成するためのベストプラクティスと、望ましい VS Code のワークフローに従うための規約を学ぶことができます。

## テストと公開

このセクションでは、高品質のVS Code拡張機能を開発するのに役立つトピックが含まれています。例えば、以下の内容を学ぶことができます。

- 拡張モジュールの統合テストを追加する方法
- 作成した拡張機能を VS Code Marketplace に公開する方法
- 拡張機能の継続的インテグレーションをセットアップする方法
