# エクステンションガイド

[Hello World](/api/get-started/your-first-extension) サンプルで Visual Studio Code Extension API の基本を学んだら、 次は実際にエクステンションを作成してみましょう。[Extension Capabilities](/api/extension-capabilities/overview) セクションでは、拡張機能でできることの概要を説明していますが、このセクションでは、特定のVS Code APIを使用する方法を説明する詳細なコードガイドとサンプルのリストが含まれています。

各ガイドやサンプルには、以下のようなものがあります。

- 徹底的にコメントされたソースコード
- サンプル拡張機能の使用方法を示すgifまたは画像。
- サンプル拡張機能を実行するための手順
- 使用されているVS Code APIのリスト
- 使用されているコントリビューションポイントのリスト
- サンプルに類似した実際のエクステンション
- APIコンセプトの説明
