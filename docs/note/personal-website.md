# 個人サイトのリンク集

私のお気に入りの個人サイトを集めました。個人的なブックマークとして使っています。

::: info 
以下のサービスを使用したウェブサイトは除外しています。

Wix / Jimdo / Weebly / Lit.link / potofu / portforiobox / folio / xfolio / Adobe Portfolio / Tumblr / ペライチ / Strikingly Ameba Ownd / (WordPress)
:::


::: details 分類の方法を考え中・・・
- 少ない ← ページ数 → 多い
- 少ない ← 情報量 → 多い
- 控えめ ←　主張 → 激しい
- 文字 ← → 絵
- 寒色 ← 色味 → 暖色
- レトロ ← 新旧 → モダン
- 雑然 ← 整頓 → 整然
- 狭い ← 広さ → 広い
:::
